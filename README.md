# Drogaria São Judas - Landing Page

Landing page moderna e responsiva desenvolvida para a **Drogaria São Judas**, com o objetivo de apresentar a farmácia de forma profissional, destacar seus serviços e facilitar o contato com os clientes.

Este projeto foi criado com foco em boas práticas de desenvolvimento front-end, organização de código, responsividade e apresentação visual, servindo também como peça de portfólio.

---

## Sobre o Projeto

A proposta deste projeto foi desenvolver uma landing page institucional para a Drogaria São Judas, utilizando tecnologias web puras, sem frameworks.

A página foi pensada para transmitir confiança, profissionalismo e proximidade com o cliente, destacando informações importantes como serviços oferecidos, localização, formas de contato e diferenciais da farmácia.

---

## Layout

O layout da landing page foi desenvolvido para ser simples, moderno e direto, valorizando a identidade visual da Drogaria São Judas.

A identidade utiliza principalmente as cores vermelho e amarelo, presentes na marca da farmácia, criando uma conexão visual com o estabelecimento físico.
